üéØ Usage Methods

Method 1: Data Attributes (Auto-init)

<div 
  data-chewie-api-url="http://localhost:8000"
  data-chewie-dapp="kamino"
  data-chewie-lang="en"
  data-chewie-theme="dark"
  data-chewie-mock="true"
></div>

Method 2: JavaScript API

<script>
  window.ChewieChat.init({
    apiUrl: 'http://localhost:8000',
    dapp: 'kamino',
    lang: 'en',
    theme: 'dark',
    position: 'bottom-right',
    mock: true,
    initialPrompts: [
      'How much can I earn on 1000 USDC?',
      'What is Kamino?'
    ],
    onEvent: (event) => {
      console.log('Chewie event:', event);
    }
  });
</script>

Available Methods

  • window.ChewieChat.init(options) - Initialize the widget
  • window.ChewieChat.destroy() - Remove the widget

‚öôÔ∏è Configuration Options

  • apiUrl (required) - Backend API endpoint
  • dapp (optional) - DApp identifier, default: 'kamino'
  • lang (optional) - Language: 'en' | 'pl', default: 'en'
  • theme (optional) - Theme: 'dark' | 'light', default: 'dark'
  • position (optional) - Position: 'bottom-right' | 'bottom-left'
  • mock (optional) - Enable mock mode for testing
  • initialPrompts (optional) - Array of suggested prompts
  • onEvent (optional) - Event callback function
